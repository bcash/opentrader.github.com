Repositório Git

Se você é um nerd aventureiro como a gente, este é o endereço do repositório git, onde estamos desenvolvendo coisas ainda mais legais no no OpenTrader. Se você quer acompanhar nossa evolução, basta clonar nosso repositório: $ git clone git://github.com/opentrader/opentrader.github.com

Se você quer contribuir no desenvolvimento do OpenTrader, você também precisará fazer um fork do nosso repositório, para que você possa fazer suas alterações, e para que possamos importá-las no repositório oficial.

* No momento apenas MySQL é suportado, porém a nova arquitetura permite extensão das funcionalidades do Model para outros datasources.

Requisitos

O Spaghetti* é construído sobre tecnologias difundidas no mercado para garantir a maior compatibilidade possível com diferentes ambientes. Contudo, alguns requisitos devem ser respeitados:

  • Servidor Apache 2.0 ou mais recente
  • Rewrite Module habilitado
  • Permissão de uso para .htaccess (AllowOverride Options FileInfo)
  • PHP 5
