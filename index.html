OpenTrader

The solution for online trading, technical analysis and automated trading.

Install

  1. Extract
    2. Extracting the source from the OpenTrader tarball is a simple matter of uncompressing, and then untarring: 
    $ tar xvzf OpenTrader-OpenTrader-*.tar.gz
    This will create a new directory under the current directory containing the source code for the distribution. You should cd into that directory before proceeding with compiling the package.
  2. Compilation
    3. Navigate to the directory with the source code and execute command compile all source files, then remove them to reduce the size of the package. 
            $ cd OpenTrader-OpenTrader-*
        $ find . -name '*.java' | xargs javac
        $ find . -name '*.java' | xargs rm
  3. Creating a manifest file
    4. $ echo -e 'Main-Class: org.opentrader.OpenTraderClass-Path: lib/seaglasslookandfeel-0.1.7.3.jar' >> manifest.mf
  4. Creating package
    5. $ jar cvfm OpenTrader.jar manifest.mf *
  5. Inclusion of necessary libraries
    6. Create a directory '/lib' and put the necessary library files (see system requirements in this file): 
    /lib/seaglasslookandfeel-0.1.7.3.jar
  6. Run OpenTrader
    7. $ java -jar OpenTrader.jar
